# Sales Performance Analysis
## Project Overview
This project analyzes retail sales data to evaluate sales, profit, and overall growth. The goal is to identify key revenue drivers, profit margin across products and customer segments, and whether the business is scaling efficiently.

## Tools Used
- Microsoft Excel (Pivot Tables, Charts)

## Key Metrics Analyzed 
- Total Sales
- Total Profit
- Profit Margin
- Average Profit Per Order

## Key Insight
March shows the highest sales ($2,750) and the strongest profit margin (15.21%), primarily driven by technology products and corporate customers. This indicates high quality sales, a stable profit margin, and that the business can expand revenue without increasing costs, and scale efficiently.

    -`Sales_Performance_Analysis.xlsx`, Excel file containing raw data, pivot tables, and a dashboard with charts.
